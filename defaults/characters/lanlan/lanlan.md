## Chibi drawing notes (reference only)
- 整体节奏比橙橙"沉一些"——身体可以稍微画得更低重心、肩部稍宽（虽然没有真正的肩，但视觉上更稳）
- **腹部水仓 + 3 颗小水滴是辨识 KO 点**，比颜色更优先保留 —— 模型最容易省略这个细节
- 手是近黑色的小肉粒，不是和身体同色 —— 描述时一定要说 `dark navy near-black stub hand`
- 表情走「从容、信赖、淡淡笑意」路线，避免大笑、惊讶、皱眉等强情绪
- 姿势优先：手按图 / 拿放大镜 / 摊开报告 / 指方向 / 给橙橙打勾 / 抱本子；不画奔跑跳跃（那是橙橙的活）
- 不画衣服、不画头发、不画耳朵

## Drift risk (critical — model defaults will betray this OC)
- ❗ 模型**完全没有这个角色的训练记忆** → 必须 `--ref characters/lanlan/lanlan.png` 每次每张
- ❗ **腹部透明窗 + 3 颗小水滴最容易丢** → prompt 必须显式写 `transparent oval window on lower belly containing three small blue painted droplets floating inside, this is a signature element and must be drawn`
- ❗ 容易画成「蓝色小人 / 蓝色精灵 / 蓝色 slime」而不是「蓝色水滴」 → 显式写 `teardrop / water-droplet body shape`
- 颜色容易漂成宝蓝、青绿、紫蓝、亮蓝 → 显式写 `deep slate dusk-blue #4A6B8A, like dusk sky, NOT bright royal blue, NOT teal, NOT purple`
- 手容易画成和身体同色 → 显式写 `dark navy near-black stub hands, distinctly darker than body`
- 容易给加帽子 / 围巾 / 眼镜 / 领结 → 显式写 `NO clothing, NO accessories, NO glasses, NO hair, NO ears`
- 描边容易变成纯黑硬线 → `dark navy watercolor line-art, soft not vector`
- 腮红容易被模型按"蓝色角色 = 蓝灰腮红"自动改色 → 显式写 `soft PINK watercolor blush circles, same as 橙橙`
- 比例容易画成 4 头身 → `chibi 2.5 head-body, head IS the body (single droplet form)`
- 和橙橙同框时，蓝蓝应略高 / 略宽（不要画反 / 不要画一样大刚好等齐）
